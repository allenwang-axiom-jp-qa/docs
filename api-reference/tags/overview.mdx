## 概述

标签管理API允许您使用标签对视频内容进行分类和组织，提高内容的可搜索性和可发现性。

## 功能特性

- **标签创建与管理**: 创建、更新、删除标签
- **标签分类**: 每个标签必须属于一个分类
- **排序管理**: 支持拖拽重排标签顺序
- **状态控制**: 启用/禁用标签
- **统计信息**: 获取标签使用统计

## 接口列表

| 方法 | 端点 | 描述 |
|------|------|------|
| GET | `/tags` | 获取标签列表 |
| GET | `/tag` | 获取单个标签详情 |
| POST | `/tag` | 创建新标签 |
| PUT | `/tag` | 更新标签信息 |
| DELETE | `/tag` | 删除标签 |
| PUT | `/tags/reorder` | 批量调整标签排序 |
| GET | `/tags/stats` | 获取标签统计 |

## Base URL

```
https://dev-admin.beauty-666.com/api/v1/site-svc-video-mgmt
```

## 数据模型

### Tag Object

| 字段 | 类型 | 描述 |
|------|------|------|
| id | int | 数据库主键ID |
| tag_id | string | 标签唯一标识符 (TAG001) |
| name | string | 标签名称 |
| category_id | int | 所属分类ID |
| category_name | string | 所属分类名称 |
| tag_type | int | 标签类型: 0=普通, 1=热门 |
| use_count | int | 使用次数 |
| video_count | int | 关联视频数量 |
| status | int | 状态: 0=禁用, 1=启用 |
| sort_order | int | 排序顺序 |
| description | string | 标签描述 |
| created_time | int | 创建时间 (Unix时间戳) |

## 快速开始

### 1. 获取所有启用的标签

```bash
curl "https://dev-admin.beauty-666.com/api/v1/site-svc-video-mgmt/tags?status=1"
```

### 2. 创建新标签

```bash
curl -X POST "https://dev-admin.beauty-666.com/api/v1/site-svc-video-mgmt/tag" \
  -H "Content-Type: application/json" \
  -d '{"name": "新标签", "categoryId": 1}'
```

### 3. 更新标签状态

```bash
curl -X PUT "https://dev-admin.beauty-666.com/api/v1/site-svc-video-mgmt/tag?tagId=1" \
  -H "Content-Type: application/json" \
  -d '{"status": 0}'
```

<Note>
  标签名称在系统中必须唯一。创建标签时，系统会自动生成唯一的 tag_id。
</Note>

<Warning>
  删除标签会自动解除该标签与所有视频的关联关系，请谨慎操作。
</Warning>
