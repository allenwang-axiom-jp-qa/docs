---
title: '标签排序'
api: 'PUT https://dev-admin.beauty-666.com/api/v1/site-svc-video-mgmt/tags/reorder'
description: '批量调整标签的排序顺序'
---

批量调整标签的排序顺序。传入标签ID数组，按照数组顺序设置标签的排序值（sortOrder）。

<Note>
  数组中第一个标签的 sortOrder 为 1，第二个为 2，以此类推。
</Note>

## 请求参数

<ParamField body="tagIds" type="array" required>
  标签ID数组，按期望的顺序排列。最少1个，最多100个。
</ParamField>

## 响应

<ResponseField name="code" type="integer">
  业务状态码（0表示成功）
</ResponseField>

<ResponseField name="msg" type="string">
  响应消息
</ResponseField>

<ResponseField name="data" type="object">
  排序结果
</ResponseField>

```bash cURL

  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "tagIds": [3, 1, 5, 2, 4]
  }'
```
