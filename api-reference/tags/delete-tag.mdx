---
title: '删除标签'
api: 'DELETE https://dev-admin.beauty-666.com/api/v1/site-svc-video-mgmt/tag'
description: '删除指定标签'
---

删除指定的标签。删除标签时，会自动解除该标签与所有视频的关联关系。

<Warning>
  删除操作不可恢复，请谨慎操作。如果只是暂时不使用，建议将标签状态设为禁用。
</Warning>

## 查询参数

<ParamField query="tagId" type="integer" required>
  要删除的标签数据库主键ID
</ParamField>

## 响应

<ResponseField name="code" type="integer">
  业务状态码（0表示成功）
</ResponseField>

<ResponseField name="msg" type="string">
  响应消息
</ResponseField>

<ResponseField name="data" type="object">
  删除结果
</ResponseField>

<RequestExample>

```bash cURL
curl -X DELETE "https://dev-admin.beauty-666.com/api/v1/site-svc-video-mgmt/tag?tagId=1" \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://dev-admin.beauty-666.com/api/v1/site-svc-video-mgmt/tag?tagId=1', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});

const data = await response.json();
```

```python Python
import requests

url = 'https://dev-admin.beauty-666.com/api/v1/site-svc-video-mgmt/tag'
params = {'tagId': 1}
headers = {
    'Authorization': 'Bearer YOUR_TOKEN'
}

response = requests.delete(url, params=params, headers=headers)
result = response.json()
```

</RequestExample>

<ResponseExample>

```json 200 删除成功
{
  "code": 0,
  "msg": "删除成功",
  "data": {
    "message": "Success"
  }
}
```

```json 400 参数错误
{
  "code": 400,
  "msg": "invalid tagId"
}
```

```json 404 标签不存在
{
  "code": 404,
  "msg": "标签不存在"
}
```

</ResponseExample>
