---
title: '获取标签列表'
api: 'GET https://dev-admin.beauty-666.com/api/v1/site-svc-video-mgmt/tags'
description: '获取标签列表，支持分页、关键词搜索和多条件筛选'
---

获取标签列表接口，支持以下功能：
- 分页查询
- 关键词模糊搜索（按名称）
- 按分类ID筛选
- 按状态筛选
- 按标签类型筛选

## 查询参数

<ParamField query="page" type="integer" default="1">
  页码，从1开始
</ParamField>

<ParamField query="pageSize" type="integer" default="20">
  每页数量，建议不超过100
</ParamField>

<ParamField query="keyword" type="string">
  搜索关键词，模糊匹配标签名称
</ParamField>

<ParamField query="categoryId" type="integer">
  按分类ID筛选
</ParamField>

<ParamField query="status" type="integer">
  状态筛选：0=禁用，1=启用
</ParamField>

<ParamField query="tagType" type="integer">
  标签类型筛选：0=普通类型，1=热门类型
</ParamField>

## 响应

<ResponseField name="code" type="integer">
  业务状态码（0表示成功）
</ResponseField>

<ResponseField name="msg" type="string">
  响应消息
</ResponseField>

<ResponseField name="data" type="object">
  响应数据
</ResponseField>

<ResponseField name="data.total" type="integer">
  总记录数
</ResponseField>

<ResponseField name="data.page" type="integer">
  当前页码
</ResponseField>

<ResponseField name="data.list" type="array">
  标签列表
</ResponseField>
