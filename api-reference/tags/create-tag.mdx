---
title: '创建标签'
api: 'POST https://dev-admin.beauty-666.com/api/v1/site-svc-video-mgmt/tag'
description: '创建新的视频标签'
---

创建一个新的视频标签。标签名称在系统中必须唯一，系统会自动生成标签标识符（tagId）。

<Note>
  标签必须关联到一个分类（categoryId 为必填项）
</Note>

## 请求参数

<ParamField body="name" type="string" required>
  标签名称，必须唯一，最大长度100字符
</ParamField>

<ParamField body="categoryId" type="integer" required>
  所属分类ID，必须是有效的分类
</ParamField>

<ParamField body="tagType" type="integer" default="0">
  标签类型：0=普通类型，1=热门类型
</ParamField>

<ParamField body="status" type="integer" default="1">
  状态：0=禁用，1=启用
</ParamField>

<ParamField body="description" type="string">
  标签描述
</ParamField>

## 响应

<ResponseField name="code" type="integer">
  业务状态码（0表示成功）
</ResponseField>

<ResponseField name="msg" type="string">
  响应消息
</ResponseField>

<ResponseField name="data" type="object">
  创建结果
</ResponseField>

<ResponseField name="data.tagId" type="integer">
  新创建标签的数据库主键ID
</ResponseField>

<RequestExample>

```bash cURL
curl -X POST "https://dev-admin.beauty-666.com/api/v1/site-svc-video-mgmt/tag" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "新标签",
    "categoryId": 1,
    "tagType": 0,
    "status": 1,
    "description": "这是一个新标签"
  }'
```

```python Python
import requests

url = 'https://dev-admin.beauty-666.com/api/v1/site-svc-video-mgmt/tag'
headers = {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
}
payload = {
    'name': '新标签',
    'categoryId': 1,
    'tagType': 0,
    'status': 1,
    'description': '这是一个新标签'
}

response = requests.post(url, json=payload, headers=headers)
result = response.json()
```
