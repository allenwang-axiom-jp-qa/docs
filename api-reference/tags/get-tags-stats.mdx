---
title: '获取标签统计'
api: 'GET https://dev-admin.beauty-666.com/api/v1/site-svc-video-mgmt/tags/stats'
description: '获取标签统计信息'
---

获取标签的整体统计信息，包括总数、各类型数量、启用数量和关联视频总数。

## 响应

<ResponseField name="code" type="integer">
  业务状态码（0表示成功）
</ResponseField>

<ResponseField name="msg" type="string">
  响应消息
</ResponseField>

<ResponseField name="data" type="object">
  统计数据
</ResponseField>

<ResponseField name="data.total" type="integer">
  标签总数
</ResponseField>

<ResponseField name="data.normalType" type="integer">
  普通类型标签数量
</ResponseField>

<ResponseField name="data.hotType" type="integer">
  热门类型标签数量
</ResponseField>

<ResponseField name="data.enabled" type="integer">
  启用状态的标签数量
</ResponseField>

<ResponseField name="data.totalVideos" type="integer">
  标签关联的视频总数（去重后）
</ResponseField>

```bash cURL
curl -X GET "https://dev-admin.beauty-666.com/api/v1/site-svc-video-mgmt/tags/stats" \
  -H "Authorization: Bearer YOUR_TOKEN"
```
