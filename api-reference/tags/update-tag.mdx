---
title: '更新标签'
api: 'PUT https://dev-admin.beauty-666.com/api/v1/site-svc-video-mgmt/tag'
description: '更新标签信息'
---

更新指定标签的信息。只需传入需要更新的字段，未传入的字段将保持原值。

## 查询参数

<ParamField query="tagId" type="integer" required>
  要更新的标签数据库主键ID
</ParamField>

## 请求参数

<ParamField body="name" type="string">
  标签名称，如果修改需确保唯一
</ParamField>

<ParamField body="categoryId" type="integer">
  所属分类ID
</ParamField>

<ParamField body="tagType" type="integer">
  标签类型：0=普通类型，1=热门类型
</ParamField>

<ParamField body="status" type="integer">
  状态：0=禁用，1=启用
</ParamField>

<ParamField body="description" type="string">
  标签描述
</ParamField>

## 响应

<ResponseField name="code" type="integer">
  业务状态码（0表示成功）
</ResponseField>

<ResponseField name="msg" type="string">
  响应消息
</ResponseField>

<ResponseField name="data" type="object">
  更新结果
</ResponseField>

<RequestExample>

```bash cURL
curl -X PUT "https://dev-admin.beauty-666.com/api/v1/site-svc-video-mgmt/tag?tagId=1" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "更新后的名称",
    "status": 0
  }'
```

```python Python
import requests

url = 'https://dev-admin.beauty-666.com/api/v1/site-svc-video-mgmt/tag'
params = {'tagId': 1}
headers = {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
}
payload = {
    'name': '更新后的名称',
    'status': 0
}

response = requests.put(url, params=params, json=payload, headers=headers)
result = response.json()
```
