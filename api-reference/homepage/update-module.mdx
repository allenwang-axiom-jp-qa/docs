---
title: '更新模块配置'
api: 'PUT https://dev-admin.beauty-666.com/api/v1/oms/homepage/modules/{id}'
description: '更新模块的标题、配置、排序或启用状态'
---

更新指定模块的配置信息。仅支持更新 `title`、`config`、`order`、`enabled` 这 4 个字段，`key` 和 `type` 为系统预设不可修改。

## Headers

<ParamField header="Authorization" type="string" required>
  JWT Bearer token，格式：`Bearer YOUR_TOKEN`
</ParamField>

<ParamField header="Content-Type" type="string" required>
  请求内容类型，必须为 `application/json`
</ParamField>

## Path Parameters

<ParamField path="id" type="integer" required>
  模块 ID，取值范围 1-9
</ParamField>

## Body

<ParamField body="title" type="string">
  模块标题，长度 1-100 个字符
</ParamField>

<ParamField body="config" type="string">
  模块配置 JSON 字符串，需要是有效的 JSON 格式
</ParamField>

<ParamField body="order" type="integer">
  显示顺序，取值范围 0-9999，数值越小越靠前
</ParamField>

<ParamField body="enabled" type="boolean">
  是否启用模块，`true` 为启用，`false` 为禁用
</ParamField>

<Note>
  所有字段都是可选的，只需要传递你想要更新的字段。未传递的字段保持原值不变。
</Note>

## Response

<ResponseField name="code" type="integer">
  响应状态码，0 表示成功
</ResponseField>

<ResponseField name="data" type="null">
  更新成功时返回 null
</ResponseField>

<ResponseField name="msg" type="string">
  响应消息
</ResponseField>

<RequestExample>

```bash cURL
curl -X PUT https://dev-admin.beauty-666.com/api/v1/oms/homepage/modules/1 \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "轮播图模块",
    "config": "{\"autoplay\":false,\"interval\":5000}",
    "order": 10,
    "enabled": false
  }'
```

```javascript JavaScript
const moduleId = 1;
const response = await fetch(`https://dev-admin.beauty-666.com/api/v1/oms/homepage/modules/${moduleId}`, {
  method: 'PUT',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    title: '轮播图模块',
    config: JSON.stringify({ autoplay: false, interval: 5000 }),
    order: 10,
    enabled: false
  })
});
const data = await response.json();
```

```python Python
import requests
import json

module_id = 1
payload = {
    'title': '轮播图模块',
    'config': json.dumps({'autoplay': False, 'interval': 5000}),
    'order': 10,
    'enabled': False
}

response = requests.put(
    f'https://dev-admin.beauty-666.com/api/v1/oms/homepage/modules/{module_id}',
    headers={
        'Authorization': 'Bearer YOUR_TOKEN',
        'Content-Type': 'application/json'
    },
    json=payload
)
data = response.json()
```

## 配置示例

<Warning>
  **重要限制：**
  - 只能更新 `title`、`config`、`order`、`enabled` 这 4 个字段
  - `key` 和 `type` 为系统预设，不可修改
  - 不支持创建和删除模块（模块固定为 9 个）
</Warning>

### Banner 模块配置

```json
{
  "autoplay": true,
  "interval": 3000,
  "showIndicator": true,
  "indicatorPosition": "bottom"
}
```

### Grid 模块配置

```json
{
  "columns": 4,
  "rows": 2,
  "gap": 10,
  "showTitle": true
}
```

### Video 模块配置

```json
{
  "layout": "list",
  "autoplay": false,
  "showDuration": true
}
```
