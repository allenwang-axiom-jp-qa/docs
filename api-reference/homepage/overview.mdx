## Introduction

The Homepage API allows you to manage homepage modules and their items for your application. The system includes 9 pre-configured modules that can be customized.

## Key Features

- **9 Pre-configured Modules**: Fixed modules including banner, hot recommend, category nav, etc.
- **Module Configuration**: Update titles, config JSON, order, and enabled status
- **Module Items CRUD**: Full create, read, update, delete support for module items
- **H5 Preview**: Optimized preview endpoint with batch query (2 queries instead of 10)
- **Pagination Support**: All list endpoints support pagination

## Pre-configured Modules

| ID | Key | Title | Type | Description |
|----|-----|-------|------|-------------|
| 1 | banner | 轮播图 | banner | Homepage carousel |
| 2 | hot_recommend | 热门推荐 | grid | Hot content grid |
| 3 | category_nav | 分类导航 | category | Category navigation |
| 4 | vip_zone | VIP专区 | vip | VIP exclusive content |
| 5 | featured_videos | 精选视频 | video | Featured videos |
| 6 | live_streaming | 直播间 | live | Live streaming |
| 7 | new_arrivals | 最新上架 | product | New products |
| 8 | discount_zone | 优惠专区 | promotion | Promotions |
| 9 | customer_service | 客服中心 | service | Customer service |

<Warning>
  Module creation and deletion are not supported. Only the 9 pre-configured modules exist in the system.
</Warning>

## Database Tables

- `home_modules` - Homepage modules (9 pre-configured)
- `home_module_items` - Homepage module items

## Performance

The H5 preview endpoint is optimized with batch queries:

- **Query Count**: 2 queries (vs 10 in naive implementation)
- **Response Time**: ~10ms (vs ~50ms)
- **Load Reduction**: 80% fewer database queries
