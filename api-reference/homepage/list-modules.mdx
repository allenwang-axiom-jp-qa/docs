---
title: '获取模块列表'
api: 'GET https://dev-admin.beauty-666.com/api/v1/oms/homepage/modules'
description: '分页获取所有首页模块列表'
---

获取所有首页模块的分页列表，支持通过 page 和 pageSize 参数控制分页。

## Headers

<ParamField header="Authorization" type="string" required>
  JWT Bearer token，格式：`Bearer YOUR_TOKEN`
</ParamField>

## Query Parameters

<ParamField query="page" type="integer" default="1">
  页码，从 1 开始
</ParamField>

<ParamField query="pageSize" type="integer" default="20">
  每页数量，最大 100
</ParamField>

## Response

<ResponseField name="code" type="integer">
  响应状态码，0 表示成功
</ResponseField>

<ResponseField name="data" type="object">
  响应数据对象

  <Expandable title="data 字段">
    <ResponseField name="list" type="array">
      模块列表数组

      <Expandable title="模块对象">
        <ResponseField name="ID" type="integer">
          模块 ID
        </ResponseField>

        <ResponseField name="key" type="string">
          模块唯一标识（系统预设，不可修改）
        </ResponseField>

        <ResponseField name="title" type="string">
          模块标题（可修改）
        </ResponseField>

        <ResponseField name="type" type="string">
          模块类型：banner、grid、category、vip、video、live、product、promotion、service
        </ResponseField>

        <ResponseField name="config" type="string">
          模块配置 JSON 字符串
        </ResponseField>

        <ResponseField name="order" type="integer">
          显示顺序（0-9999）
        </ResponseField>

        <ResponseField name="enabled" type="boolean">
          是否启用
        </ResponseField>

        <ResponseField name="CreatedAt" type="string">
          创建时间（ISO 8601 格式）
        </ResponseField>

        <ResponseField name="UpdatedAt" type="string">
          更新时间（ISO 8601 格式）
        </ResponseField>
      </Expandable>
    </ResponseField>

    <ResponseField name="total" type="integer">
      总记录数
    </ResponseField>

    <ResponseField name="page" type="integer">
      当前页码
    </ResponseField>

    <ResponseField name="pageSize" type="integer">
      每页数量
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="msg" type="string">
  响应消息
</ResponseField>

```python Python
import requests

params = {
    'page': 1,
    'pageSize': 10
}

response = requests.get(
    'https://dev-admin.beauty-666.com/api/v1/oms/homepage/modules',
    headers={'Authorization': 'Bearer YOUR_TOKEN'},
    params=params
)
data = response.json()
```

## 模块类型说明

| Key | 标题 | 类型 | 描述 |
|-----|------|------|------|
| banner | 轮播图 | banner | 首页顶部轮播 |
| hot_recommend | 热门推荐 | grid | 热门内容网格 |
| category_nav | 分类导航 | category | 分类快捷入口 |
| vip_zone | VIP专区 | vip | 会员专属内容 |
| featured_videos | 精选视频 | video | 精选视频列表 |
| live_streaming | 直播间 | live | 直播频道 |
| new_arrivals | 最新上架 | product | 新品推荐 |
| discount_zone | 优惠专区 | promotion | 促销活动 |
| customer_service | 客服中心 | service | 客服入口 |

<Warning>
  系统预设了 9 个模块，不支持创建和删除模块。只能修改已有模块的配置。
</Warning>
