---
title: '获取统计概览'
api: 'GET https://dev-admin.beauty-666.com/api/v1/oms/homepage/overview'
description: '获取首页模块的统计信息，包括总数、启用数和禁用数'
---

获取所有首页模块的统计概览数据，用于管理后台仪表板展示。

## Headers

<ParamField header="Authorization" type="string" required>
  JWT Bearer token，格式：`Bearer YOUR_TOKEN`
</ParamField>

## Response

<ResponseField name="code" type="integer">
  响应状态码，0 表示成功
</ResponseField>

<ResponseField name="data" type="object">
  统计数据对象

  <Expandable title="data 字段">
    <ResponseField name="total_count" type="integer">
      模块总数（固定为 9）
    </ResponseField>

    <ResponseField name="enabled_count" type="integer">
      已启用的模块数量
    </ResponseField>

    <ResponseField name="disabled_count" type="integer">
      已禁用的模块数量
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="msg" type="string">
  响应消息
</ResponseField>

<RequestExample>

```bash cURL
curl https://dev-admin.beauty-666.com/api/v1/oms/homepage/overview \

```

## 使用场景

此接口通常用于：
- 管理后台首页仪表板数据展示
- 快速了解首页模块配置状态
- 监控哪些模块已启用/禁用

<Info>
  **系统说明**：首页系统预设了 9 个固定模块类型（banner、navigation、video_list、product_grid、swiper、icon_nav、notice、coupon、ad_banner），不能新增或删除模块，只能启用/禁用和编辑配置。
</Info>
