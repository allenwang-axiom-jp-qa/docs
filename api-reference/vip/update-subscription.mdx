---
title: '更新套餐'
api: 'PUT https://dev-admin.beauty-666.com/api/v1/oms/subscriptions/{id}'
description: '更新指定 ID 的会员套餐配置，支持部分字段更新'
---

更新指定套餐的配置信息，所有字段都是可选的，只更新传递的字段。

## 请求头

<ParamField header="Authorization" type="string" required>
  JWT Bearer token
</ParamField>

<ParamField header="Content-Type" type="string" required>
  必须为 `application/json`
</ParamField>

## 路径参数

<ParamField path="id" type="integer" required>
  套餐 ID
</ParamField>

## 请求体

所有字段都是可选的，只传需要更新的字段：

<ParamField body="name" type="string">
  套餐名称
</ParamField>

<ParamField body="duration_type" type="string">
  时长类型
</ParamField>

<ParamField body="duration_days" type="integer">
  时长天数
</ParamField>

<ParamField body="price_cny" type="string">
  人民币价格（DECIMAL 字符串格式）
</ParamField>

<ParamField body="original_price_cny" type="string">
  人民币原价（DECIMAL 字符串格式）
</ParamField>

<ParamField body="price_usdt" type="string">
  USDT 价格（DECIMAL 字符串格式）
</ParamField>

<ParamField body="original_price_usdt" type="string">
  USDT 原价（DECIMAL 字符串格式）
</ParamField>

<ParamField body="benefits" type="array">
  会员权益列表（字符串数组）
</ParamField>

<ParamField body="status" type="integer">
  状态：0=下架，1=上架
</ParamField>

<ParamField body="sort_order" type="integer">
  排序值
</ParamField>

<ParamField body="description" type="string">
  备注说明
</ParamField>

<RequestExample>

```bash cURL
# 仅更新价格和状态
curl -X PUT https://dev-admin.beauty-666.com/api/v1/oms/subscriptions/1 \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "月度VIP会员（限时优惠）",
    "price_cny": "19.90",
    "status": 1
  }'
```

```javascript JavaScript
// 仅更新套餐名称
const subscriptionId = 1;
const response = await fetch(
  `https://dev-admin.beauty-666.com/api/v1/oms/subscriptions/${subscriptionId}`,
  {
    method: 'PUT',
    headers: {
      'Authorization': 'Bearer YOUR_TOKEN',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      name: '月度VIP会员（限时优惠）'
    })
  }
);
```

```python Python
import requests

subscription_id = 1
payload = {
    'price_cny': '19.90',  # 字符串格式
    'status': 0  # 下架
}

response = requests.put(
    f'https://dev-admin.beauty-666.com/api/v1/oms/subscriptions/{subscription_id}',
    headers={
        'Authorization': 'Bearer YOUR_TOKEN',
        'Content-Type': 'application/json'
    },
    json=payload
)
```

</RequestExample>

<ResponseExample>

```json 200 成功
{
  "code": 0,
  "data": null,
  "msg": "更新套餐成功"
}
```

```json 404 套餐不存在
{
  "code": 7,
  "data": null,
  "msg": "套餐不存在或数据未变更"
}
```

</ResponseExample>

<Info>
  部分字段更新示例：
  - 仅更新价格：只传 `price_cny` 或 `price_usdt`
  - 上架/下架：只传 `status`
  - 修改排序：只传 `sort_order`
  - 其他字段保持不变
</Info>
