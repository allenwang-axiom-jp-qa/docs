---
title: '获取套餐详情'
api: 'GET https://dev-admin.beauty-666.com/api/v1/oms/subscriptions/{id}'
description: '根据 ID 获取单个会员套餐的详细信息'
---

根据套餐 ID 获取单个会员套餐的完整详细信息，包括所有配置和计算字段。

## 请求头

<ParamField header="Authorization" type="string" required>
  JWT Bearer token，格式：`Bearer YOUR_TOKEN`
</ParamField>

## 路径参数

<ParamField path="id" type="integer" required>
  套餐 ID
</ParamField>

## 响应

<ResponseField name="code" type="integer">
  响应状态码，0 表示成功
</ResponseField>

<ResponseField name="data" type="object">
  套餐详情对象（字段说明同列表接口）
</ResponseField>

<ResponseField name="msg" type="string">
  响应消息
</ResponseField>

<RequestExample>

```bash cURL
curl https://dev-admin.beauty-666.com/api/v1/oms/subscriptions/1 \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const subscriptionId = 1;
const response = await fetch(
  `https://dev-admin.beauty-666.com/api/v1/oms/subscriptions/${subscriptionId}`,
  {
    headers: {
      'Authorization': 'Bearer YOUR_TOKEN'
    }
  }
);
const data = await response.json();
```

```python Python
import requests

subscription_id = 1
response = requests.get(
    f'https://dev-admin.beauty-666.com/api/v1/oms/subscriptions/{subscription_id}',
    headers={'Authorization': 'Bearer YOUR_TOKEN'}
)
data = response.json()
```
