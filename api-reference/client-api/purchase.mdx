# 影音购买服务 API

提供消费相关功能的 API 接口。

## 功能模块

### 1. 会员相关

| 接口 | 说明 |
|------|------|
| 获取会员方案列表 | 获取所有会员方案（月度/季度/年度） |
| 查询会员状态 | 查询当前用户的会员状态和有效期 |

### 2. 钻石充值

| 接口 | 说明 |
|------|------|
| 获取钻石方案列表 | 获取各档位的钻石充值方案 |

### 3. 影片购买

| 接口 | 说明 |
|------|------|
| 创建购买订单 | 使用钻石购买影片 |
| 查询已购列表 | 获取用户已购买的影片列表 |

<Warning>
影片购买仅支持 **钻石支付** 方式。
</Warning>

### 4. 订单查询

| 接口 | 说明 |
|------|------|
| 查询订单详情 | 根据订单号查询订单详情 |
| 查询订单列表 | 支持按分类、状态、日期范围筛选 |

## 订单状态说明

### 订单状态 (order_status)
| 值 | 说明 |
|----|------|
| 0 | 待支付 |
| 1 | 已完成 |
| 2 | 已取消 |
| 3 | 已退款 |

### 支付状态 (payment_status)
| 值 | 说明 |
|----|------|
| 0 | 待支付 |
| 1 | 支付中 |
| 2 | 已支付 |
| 3 | 支付失败 |
| 4 | 已退款 |

## 订单分类说明

| 分类 | 说明 |
|------|------|
| CONSUMPTION | 消费纪录（会员购买 + 钻石购买影片） |
| DIAMOND_RECHARGE | 钻石充值纪录 |
| DEPOSIT_WITHDRAW | 充提纪录（存款 + 取款） |
