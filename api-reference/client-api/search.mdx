# 影片搜索 API

提供视频搜索和筛选功能的 API 接口。

## 接口列表

| 接口 | Method | Path | 说明 |
|------|--------|------|------|
| 搜索视频 | GET | `/api/v1/search/videos` | 根据条件搜索视频 |
| 热门搜索 | GET | `/api/v1/search/hot` | 获取热门搜索词 |
| 搜索建议 | GET | `/api/v1/search/suggest` | 获取搜索建议 |

## 搜索功能

### 支持的搜索方式

| 方式 | 参数 | 说明 |
|------|------|------|
| 关键词搜索 | `keyword` | 匹配标题、描述 |
| 标签搜索 | `tag_id` | 按标签 ID 筛选 |
| 分类搜索 | `category_id` | 按分类 ID 筛选 |

### 筛选参数

| 参数 | 类型 | 说明 |
|------|------|------|
| duration_min | integer | 最小时长（秒） |
| duration_max | integer | 最大时长（秒） |
| publish_date | string | 发布日期筛选（today/week/month） |
| quality | string | 清晰度（720p/1080p/4k） |

### 排序方式

| 值 | 说明 |
|----|------|
| latest | 最新发布 |
| views | 最多观看 |
| likes | 最多点赞 |
| favorites | 最多收藏 |

## 请求示例

### 搜索视频

```bash
curl -X GET "https://dev-client.beauty-666.com/api/v1/search/videos?keyword=test&page=1&page_size=20&sort=latest" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
```

### 请求参数

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| keyword | string | 否 | 搜索关键词 |
| category_id | integer | 否 | 分类 ID |
| tag_id | integer | 否 | 标签 ID |
| sort | string | 否 | 排序方式，默认 `latest` |
| page | integer | 否 | 页码，默认 1 |
| page_size | integer | 否 | 每页数量，默认 20，最大 100 |
