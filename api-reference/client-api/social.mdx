# 社交互动服务 API

提供社交互动功能的 API 接口。

## 功能模块

### 1. 评论功能

| 接口 | 说明 |
|------|------|
| 发表评论 | 发表顶层评论 |
| 发表回复 | 发表嵌套评论（回复其他评论） |
| 获取评论列表 | 获取影片的顶层评论列表 |
| 获取回复列表 | 获取某条评论的回复列表 |
| 删除评论 | 删除自己发表的评论 |

### 2. 点赞功能

| 接口 | 说明 |
|------|------|
| 影片点赞/取消 | Toggle 切换点赞状态 |
| 评论点赞/取消 | Toggle 切换评论点赞状态 |
| 批量检查点赞状态 | 批量查询用户是否已点赞 |

### 3. 收藏功能

| 接口 | 说明 |
|------|------|
| 收藏/取消收藏 | Toggle 切换收藏状态 |
| 获取收藏列表 | 获取用户的收藏影片列表 |
| 批量检查收藏状态 | 批量查询用户是否已收藏 |

<Note>
点赞和收藏接口采用 Toggle 模式，同一接口可切换状态。
</Note>
