## 认证方式

所有用户端 API 都需要通过 JWT Token 进行认证：

```bash
curl -X GET "https://api.example.com/api/v1/client/..." \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json"
```

## 基础响应格式

```json
{
  "code": 0,
  "msg": "success",
  "data": {
    // 响应数据
  }
}
```

### 错误码说明

| 错误码 | 说明 |
|--------|------|
| 0 | 成功 |
| 401 | 未授权 |
| 403 | 禁止访问 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |
