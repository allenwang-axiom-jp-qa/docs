# CMS 网站客户端 API

提供用户端基础功能的 API 接口。

## 功能模块

### 首页模块
- 获取首页配置
- 获取推荐内容
- 获取轮播图

### 视频模块
- 视频列表查询
- 视频详情获取
- 视频播放相关

### 用户模块
- 用户资料查询
- 用户设置更新

<Note>
所有接口均需要携带有效的 JWT Token 进行认证。
</Note>
