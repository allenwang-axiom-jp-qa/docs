---
title: '创建币种'
api: 'POST https://dev-admin.beauty-666.com/api/v1/oms/exchange-rates'
description: '创建新的币种，汇率需在基准汇率 ±5% 范围内'
---

创建新的币种配置。系统会对汇率进行 ±5% 范围校验，超出范围将阻止操作。

## 请求头

<ParamField header="Authorization" type="string" required>
  JWT Bearer token，格式：`Bearer YOUR_TOKEN`
</ParamField>

<ParamField header="Content-Type" type="string" required>
  `application/json`
</ParamField>

## 请求体

<ParamField body="code" type="string" required>
  币种代码（2-10 字符，自动转大写，如 USD、USDT）
</ParamField>

<ParamField body="name" type="string" required>
  币种名称（1-50 字符，如"美元"、"泰达币"）
</ParamField>

<ParamField body="symbol" type="string" required>
  货币符号（1-5 字符，如 $、¥、₮）
</ParamField>

<ParamField body="currency_type" type="integer" required>
  币种类型：
  - `0` - 法币
  - `1` - 加密货币
</ParamField>

<ParamField body="exchange_rate" type="string" required>
  对 CNY 的汇率（必须 > 0，且在基准汇率 ±5% 范围内）
</ParamField>

<ParamField body="update_mode" type="integer" required>
  更新模式：
  - `0` - 手动更新
  - `1` - 自动同步
</ParamField>

## 响应

<ResponseField name="code" type="integer">
  响应状态码，0 表示成功，7 表示业务错误
</ResponseField>

<ResponseField name="data" type="object">
  创建成功的币种对象

  <Expandable title="币种对象">
    <ResponseField name="id" type="integer">
      新创建的币种 ID
    </ResponseField>
    <ResponseField name="code" type="string">
      币种代码
    </ResponseField>
    <ResponseField name="name" type="string">
      币种名称
    </ResponseField>
    <ResponseField name="symbol" type="string">
      货币符号
    </ResponseField>
    <ResponseField name="currency_type" type="integer">
      币种类型
    </ResponseField>
    <ResponseField name="exchange_rate" type="string">
      汇率
    </ResponseField>
    <ResponseField name="update_mode" type="integer">
      更新模式
    </ResponseField>
    <ResponseField name="last_updated_at" type="string">
      更新时间
    </ResponseField>
    <ResponseField name="status" type="integer">
      状态（默认为 1=启用）
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="msg" type="string">
  响应消息
</ResponseField>

<Warning>
  **汇率范围校验（±5%）**

  系统对已配置基准汇率的币种进行 ±5% 范围校验：

  | 币种 | 基准汇率 | 允许范围 |
  |------|---------|---------|
  | USDT | 7.2 | 6.84 ~ 7.56 |
  | USD | 7.2 | 6.84 ~ 7.56 |
  | EUR | 7.8 | 7.41 ~ 8.19 |
  | JPY | 0.048 | 0.0456 ~ 0.0504 |
  | BRL | 1.4 | 1.33 ~ 1.47 |

  超出范围的汇率将被**阻止创建**，请确认输入是否正确。
</Warning>
