---
title: '更新币种'
api: 'PUT https://dev-admin.beauty-666.com/api/v1/oms/exchange-rates/{id}'
description: '更新币种信息，汇率需在基准汇率 ±5% 范围内'
---

更新指定币种的配置信息。所有字段均为可选，仅更新传递的字段。币种代码（code）创建后不可修改。

## 请求头

<ParamField header="Authorization" type="string" required>
  JWT Bearer token，格式：`Bearer YOUR_TOKEN`
</ParamField>

<ParamField header="Content-Type" type="string" required>
  `application/json`
</ParamField>

## 路径参数

<ParamField path="id" type="integer" required>
  币种 ID
</ParamField>

## 请求体（所有字段可选）

<ParamField body="name" type="string">
  币种名称（1-50 字符）
</ParamField>

<ParamField body="symbol" type="string">
  货币符号（1-5 字符）
</ParamField>

<ParamField body="currency_type" type="integer">
  币种类型：0=法币，1=加密货币
</ParamField>

<ParamField body="exchange_rate" type="string">
  对 CNY 的汇率（必须在基准汇率 ±5% 范围内）
</ParamField>

<ParamField body="update_mode" type="integer">
  更新模式：0=手动，1=自动同步
</ParamField>

<ParamField body="status" type="integer">
  状态：0=停用，1=启用
</ParamField>

## 响应

<ResponseField name="code" type="integer">
  响应状态码，0 表示成功，7 表示业务错误
</ResponseField>

<ResponseField name="data" type="object">
  更新后的币种对象
</ResponseField>

<ResponseField name="msg" type="string">
  响应消息
</ResponseField>

<RequestExample>

```bash cURL
# 仅更新汇率
curl -X PUT "https://dev-admin.beauty-666.com/api/v1/oms/exchange-rates/10" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "exchange_rate": "7.3"
  }'
```

```javascript JavaScript
const currencyId = 10;
const response = await fetch(
  `https://dev-admin.beauty-666.com/api/v1/oms/exchange-rates/${currencyId}`,
  {
    method: 'PUT',
    headers: {
      'Authorization': 'Bearer YOUR_TOKEN',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      exchange_rate: '7.3',
      name: '泰达币 (USDT)'
    })
  }
);
const data = await response.json();
```

```python Python
import requests

currency_id = 10
response = requests.put(
    f'https://dev-admin.beauty-666.com/api/v1/oms/exchange-rates/{currency_id}',
    headers={
        'Authorization': 'Bearer YOUR_TOKEN',
        'Content-Type': 'application/json'
    },
    json={
        'exchange_rate': '7.3'
    }
)
data = response.json()
```

<Warning>
  **基准货币保护**

  CNY（人民币）是基准货币，其汇率固定为 1，不可修改。尝试修改 CNY 的汇率将被拒绝。
</Warning>

<Info>
  **币种代码不可修改**

  币种代码（code）在创建后不可更改。如需更换币种代码，请删除后重新创建。
</Info>
